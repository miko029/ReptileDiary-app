# 爬宠管家功能页面清单

本文档整理当前项目需要覆盖的移动端功能页，用于后续 Figma 设计、原型串联和开发排期。

## 一级 Tab 页面

| 页面 | 路由 | 主要功能 |
| --- | --- | --- |
| 首页 | `pages/index/index` | 今日概览、健康/待喂/逾期统计、待喂宠物、单只喂食打卡、按爬柜批量喂食打卡、本周喂食安排 |
| 宠物列表 | `pages/pets/pets` | 宠物档案列表、状态筛选、进入宠物详情、新增宠物、进入爬柜管理 |
| 日历 | `pages/calendar/calendar` | 月历查看喂食计划、查看选中日期安排、补记/添加喂食、进入宠物详情 |
| 我的 | `pages/profile/profile` | 应用信息、宠物统计、提醒设置、默认喂食周期、健康管理、数据导出、关于产品 |

## 核心业务页面

| 页面 | 路由 | 主要功能 |
| --- | --- | --- |
| 宠物详情 | `pages/pet-detail/pet-detail` | 查看单只宠物档案、喂食信息、喂食打卡、编辑宠物、删除宠物、查看喂食记录和健康记录 |
| 新增/编辑宠物 | `pages/pet-form/pet-form` | 新增或编辑宠物名称、分类、品种、喂食周期、上次喂食、健康状态、爬柜位置、备注 |
| 爬柜管理 | `pages/cabinet-management/cabinet-management` | 新增/编辑/删除爬柜，维护柜位，查看柜位占用情况 |
| 健康管理 | `pages/health-management/health-management` | 按健康状态筛选宠物，查看健康统计，给宠物添加健康记录 |
| 健康记录表单 | `pages/health-record-form/health-record-form` | 添加体重、蜕皮、食欲、活动状态、健康状态、备注等记录 |

## 设置与辅助页面

| 页面 | 路由 | 主要功能 |
| --- | --- | --- |
| 提醒设置 | `pages/reminder-settings/reminder-settings` | 开启喂食提醒、默认提醒时间、逾期提醒、提前一天提醒 |
| 默认喂食周期 | `pages/feed-cycle-settings/feed-cycle-settings` | 设置新增宠物默认喂食周期 |
| 数据导出 | 当前为弹窗/预留入口 | 后续可导出 CSV 或 JSON 数据 |
| 关于产品 | 当前为弹窗/预留入口 | 展示版本信息、产品说明、使用说明 |

## 建议设计优先级

| 优先级 | 页面 | 原因 |
| --- | --- | --- |
| P0 | 首页 | 用户每天打开最常用，需要承载待喂、逾期、快速打卡和整柜打卡 |
| P0 | 宠物列表 | 宠物管理入口，决定建档和查找效率 |
| P0 | 宠物详情 | 单只宠物的核心信息和记录沉淀页 |
| P0 | 新增/编辑宠物 | 建档闭环核心页，字段多，需要重点优化表单体验 |
| P0 | 爬柜管理 | 项目特色能力，影响整柜打卡和宠物位置管理 |
| P1 | 日历 | 喂食计划和历史记录的时间视图，适合做强视觉页面 |
| P1 | 我的 | 设置、统计和应用信息入口 |
| P1 | 健康管理 | 健康状态筛选和健康记录入口 |
| P1 | 健康记录表单 | 补充健康数据，用于长期饲养记录 |
| P2 | 提醒设置 | 影响留存和任务提醒，但第一版可简化 |
| P2 | 默认喂食周期 | 设置项较轻，可作为辅助页处理 |
